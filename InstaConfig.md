# INSTALAÇÕES E CONFIGURAÇÕES

## MAVEN e GIT

**1. Instalação e versão**

- Atualização dos pacotes:  
  ```
  $ sudo apt-get update
  ```
- Instalação e versão do Maven:
  ```
  $ sudo apt-get -y install maven
  $ mvn --version
  ```
- Instalação e versão do Git:
  ```
  $ sudo apt-get install git
  $ git --version
  ```
